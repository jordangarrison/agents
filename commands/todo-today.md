# Today's Tasks

Show the user their tasks that are due today.

## Implementation

Use the Todoist MCP server to fetch tasks filtered by today's date.

## Presentation

Present results organized by:
1. Overdue tasks (if any)
2. High priority (p1, p2)
3. Normal priority (p3, p4)

Include project names and any relevant labels.
