# Complete Task

Mark a task as complete.

## Arguments

Use "$ARGUMENTS" to identify the task by:
- Task name/content (fuzzy match)
- Task ID (if provided)

If multiple tasks match, show options and ask user to confirm.

## Implementation

Use the Todoist MCP server to complete the task.
Show the completed task and any remaining related tasks.
